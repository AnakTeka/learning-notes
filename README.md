# PySorting

A Python program for different sorting algorithms

## Sorting Algorithms:

* Quick sort
* Insertion Sort
* Bubble sort
* Selection sort
* Shell sort
* Binary Tree sort
* Merge sort
* Gnome sort
* Cocktail sort

## How-To

It is compatible in both Python 2.x and Python 3.x, it runs on the standard in-build library, no external dependencies need.
